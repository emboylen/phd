## Methodology

### BERTopic Algorithm

BERTopic is a topic modeling technique that leverages transformer-based language models and class-based TF-IDF to create dense clusters and extract meaningful topics.

**Key Steps:**

1. **Document Embedding:** Papers converted to semantic embeddings using sentence transformers
2. **Dimensionality Reduction:** UMAP used to reduce embedding dimensions
3. **Clustering:** HDBSCAN algorithm groups similar papers
4. **Topic Representation:** Class-based TF-IDF extracts representative keywords
